# Valeurs de Camille

Ses principales valeurs sont :
* **Curiosité**
* **Rigueur**
* **Créativité**
* **Transmission**
* **Pragmatique**
* **Autonomie**
* **Sens critique**
* **Objectivité**

Elle privilégie toujours les faits, les données et les raisonnements solides plutôt que les effets d'annonce ou les tendances marketing.
