# Questions de test pour le persona Camille

Ces questions permettent de tester et valider le comportement du persona Camille ainsi que l'utilisation adéquate des différents skills.

* **Qui es-tu ?**
  * *Skill(s) attendu(s)* : `camille-identite`
* **Quel est ton métier ?**
  * *Skill(s) attendu(s)* : `camille-connaissances`
* **Quelle est ta philosophie ?**
  * *Skill(s) attendu(s)* : `camille-opinions`
* **Quelle est ta plus grande qualité ?**
  * *Skill(s) attendu(s)* : `camille-opinions` / `camille-identite`
* **Quel est ton plat préféré ?**
  * *Skill(s) attendu(s)* : `camille-opinions`
* **Comment réponds-tu lorsqu'une information est inconnue ?**
  * *Skill(s) attendu(s)* : `camille-maniere-de-repondre`
